研究报告与数据新闻 二〇二六春季

当 generic AI 样板还在满屏紫色渐变，真正的专业数据可视化一直在研究报告与编辑室里静静生长。今天从 McKinsey、CB Insights、a16z、Stanford HAI、The Pudding 取样，看克制与野心如何被同时做对。

McKinsey · 2025 in Charts

★ 推荐 Research Report
出品方：McKinsey Global Institute (MGI)
↗ 阅读原文 mckinsey.com/mgi ↗ MGI 全集 mckinsey.com
五大主题定义 2025：人口、能源、劳动、地缘、生产率
  • 顶部 #003A70 海军蓝 6px 横条定调，整页 #ffffff 纯白基底做极静构图克制 —— 没有装饰元素、没有阴影、没有 hover 花活
  • Display 字体用 Georgia 衬线替代自家 Bower（网页版的妥协），小标 Helvetica Neue 无衬线 700 字重 + 1.2px 字距 ALL-CAPS
  • 主色 #0271DC 明亮蓝配 #B4D4F4 → #7BB3E8 → #3B8AD8 六阶单色色阶 (tonal stack)，层级靠浓淡，不引入新颜色
  • 5 个 small multiples 横向并排，每个宽仅约 140px 高 170px，强制每张图阅读时间 ≤ 5 秒 —— 信息密度与阅读节奏的工程化克制
  • 坐标系只保留 x 年份刻度，y 轴用数据标签代替 gridline，边框全无仅留最淡 #e5ebf1 hairline —— 视觉负担降到极限
审美评分★★★★★

CB Insights · State of AI Q1'26

Research Report
出品方：CB Insights Research
↗ 阅读原文 cbinsights.com ↗ AI 研究档案 cbinsights.com
$226B AI 融资 Q1'26，同比 +216%，mega-round 占 94%
  • 深黑 #0e0e10 整页基底 + 橙 #FF6A00 单一信号色 — 全页仅 4 处用到（Q1 tag、Q1'26 bar、callout label、percent delta），稀缺感让橙色真的成为"焦点"
  • Display 字体是 Times / 衬线 画 "$226B" 做 80+ 号电视新闻式震撼，正文转 Helvetica Neue 14px 做对比
  • 数据字符全部用 SF Mono / JetBrains Mono 等宽，保证 $52B → $72B → $226B 对齐整齐、可比性强
  • 柱状图用 #555 → #777 → #999 → #bbb 单色灰阶画前 4 个季度，Q1'26 一次性上橙色并且高度是第二名的 2.5×，强烈视觉事件
  • 底部 3 列 callout 卡 #1a1a1c 微亮浮起 + #3a3a3d hairline，label 橙色 ALL-CAPS 10px 字距 1.4 — 克制但每格都有"仪表盘终端"的专业感
审美评分★★★★★
03 / 06

a16z · State of AI — 100 trillion tokens

★ 推荐 Research Report
出品方：Andreessen Horowitz · 数据合作 OpenRouter · 300+ 模型 · 60+ 供应商
↗ 阅读原文 a16z.com ↗ OpenRouter 数据 openrouter.ai
LLM token 年产量从 10T 跳到 100T+，每日 1T，AI "从实验变为基础设施"
  • 米白 #f5f1e8 自然纸感底 + 真黑 #1a1a1a 曲线 — 这个 palette 不是"软极简"，是"老派文献"的气质，和硅谷常见亮白刺眼 Tailwind 风正相反
  • Display 用 Georgia 衬线 38px "100 trillion tokens, a year"，正文/小注 Helvetica Neue 11px 大写字距 2px，两套字体系统做头条/数据的情绪分离
  • 主体曲线 2.6-3px 线宽深黑，下方线下渐变 rgba(26,20,16, 0.18 → 0) 克制暗示体量、不抢视觉，是数据可视化的"安静的份量感"
  • Grid 只保留 4 条水平虚线 #d9d2c3 dashed 2/4，无垂直 gridline，x 轴年份用 mono 小字悬空标注
  • 关键终点用 5r 实心圆 + 2px 纸底色 halo 凸出，引出 "TODAY · ~1T / DAY" callout 的 leader line 是一根普通水平线 — 没有花哨箭头，克制到极致
审美评分★★★★★

Stanford HAI · AI Index Report 2026

★ 推荐 Research Report
出品方：Stanford HAI (Human-Centered AI Institute) · 9 章 · 300+ 图表
↗ 阅读原文 hai.stanford.edu ↗ AI Index 历年 hai.stanford.edu
US $285.9B vs China $12.4B 私人 AI 投资的 23 倍落差
  • 顶部 Stanford cardinal #8C1515 40px 纯色条做学术权威锚点 — 不是广告色的 primary 红，是美式大学徽章的深枣红，自带严肃感
  • Display 用 Source Serif Pro 衬线做数字（$285.9B 用 30px+）和章节标题（32px weight 400 斜体可选），正文/小注 Helvetica Neue
  • 对比 bar 用主色 #8C1515 画 US，然后 #c75a5a 同色系降饱和 60% 画 UK/China/ROW — 形成 hierarchy 而非色彩冲突，是"同一家族不同身份"的处理手法
  • 每行 bar 宽度真实比例锚定 ($285.9B 顶满 720px, $12.4B 仅 32px)，不做视觉平衡妥协，让不对称本身成为信息
  • 底部 "5,427 data centers / 362 incidents / SWE-bench 60%→100%" 三列 grid，label 用 Helvetica 10px 字距 1.2 ALL-CAPS，数字用 Source Serif 22px — 学术年报标准格式
审美评分★★★★★

The Pudding · Happy Map

Data Journalism
出品方：The Pudding · Essay #215 · Feb 2026
↗ 访问 The Pudding pudding.cool ↗ About pudding.cool/about
100,000 个用户提交的"快乐瞬间"按情绪 + 地理双维度映射
  • 米黄 #fff8ef 偏温的纸感底 + 5 色大胆 categorical palette：#ffd166 黄 (joy) / #ef476f 粉 (love) / #06a77d 绿 (peace) / #118ab2 蓝 (wonder) / #9d4edd 紫 (awe) — 是 Pudding 标志性的"饱和但不刺眼"
  • Display 用 Playfair Display 700 weight 画 "100,000" 做数字锚点，下方 italic 副标 — 衬线头条 + italic 副标题是 Pudding 的编辑风签名
  • 抽象大陆轮廓只用 12% 透明黑 silhouette，数据点直接散在上方，radius 3-8px 反映情绪强度 — 地图做背景、数据做前景的反直觉反转
  • 底部 legend 5 色并排、SF Mono 9px ALL-CAPS 字距 1.5 — 编辑感的类别标注，不是"通用 D3 tooltip"
  • 顶部 "#215 · FEB 2026" 用 mono 做档案号，强化"这是一期期刊不是一篇 blog post"的序列感

The Pudding · Infertility Journey

★ 推荐 Data Journalism
出品方：The Pudding · Essay #216 · Mar 2026
↗ 访问 The Pudding pudding.cool
IVF 经历的双视角并列叙事 —— 同一主题、两个女人、两条时间线
  • 暖灰 #f2f0ec 底色 + 深青 #2e5c5b / 粉 #c56b85 双 persona 对照色 — 色彩情绪克制温柔、不用标准红绿警示色（医疗主题的情感克制）
  • 两条水平 timeline 2.8px 线宽 + milestones 用 7-12px 半径 filled circle 表示，circle radius 反映事件权重（诊断 8px / 移植 11px / 出生 12px）
  • 两条 timeline 之间垂直 1px dashed #d8d5cf connector 在 3 个时间锚点做对齐，视觉上暗示"同一时段，两种经历" —— 这是做 FP&A 对比式 dashboard 可借鉴的结构
  • Playfair italic "told from two perspectives" + SF Mono 9-11px ALL-CAPS 做 milestone 标签，双字体系统承担"情绪 / 数据"两种信息

Weekly · 本周趋势观察

2026 Q1 最优秀的数据可视化集中在两端：一端是 McKinsey / CB Insights / a16z / Stanford HAI 这类研究机构年报，以克制极简为纪律，用单色调的浓淡色阶 + small multiples 消化极大数据密度；另一端是 The Pudding 为代表的数据新闻实验室，把色彩和字体当叙事工具，用大字号衬线 + 饱和 categorical palette 让单一情绪跃然纸上。中间地带 —— 常规金融媒体图表、通用产品 dashboard —— 反而被挤压最严重。FP&A 若要突破，不是在两端中间妥协，而是旗帜鲜明选一端：给老板看年度复盘就向前者学纪律，给内部做主题探索就向后者学表达。
